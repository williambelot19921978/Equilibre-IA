```
YYYY-MM-DD-nom-feature.md
```

Exemple : `2026-07-20-sprint-a1-interfaces-pipeline.md`

## Template

Utiliser [`../templates/architecture-review.template.md`](../templates/architecture-review.template.md).

## Statuts

| Statut | Signification |
|--------|---------------|
| `draft` | Revue en cours |
| `approved` | Validé humainement — GO DEV |
| `approved-with-reserves` | GO avec mitigations documentées |
| `rejected` | NO GO |
| `deferred` | Reporté |
